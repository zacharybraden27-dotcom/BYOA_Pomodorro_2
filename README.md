## How to Use

1. Open `index.html` in your web browser
2. Enter what you want to focus on in the goal input field
3. Click **Start** to begin the 25-minute timer
4. Click **Pause** if you need to take a break
5. Click **Reset** to restart the timer
6. When the timer completes, you'll get a notification to take a break!
